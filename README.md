# Smooth Project

Projeto experimental de front-end criado para explorar **smooth scrolling** e **animações fluidas** no browser. O objetivo é estudar como animações baseadas em scroll se comportam em conjunto, com foco em timing, performance e experiência do usuário.

## Visão Geral

Este projeto combina **GSAP** e **Locomotive Scroll** para criar uma experiência de rolagem suave e visualmente agradável. Ele utiliza conteúdos simples, como pequenos textos e imagens de animais, permitindo total foco no movimento, nas transições e no comportamento do scroll.

## Objetivo

* Experimentar comportamentos de smooth scroll
* Testar animações acionadas por scroll e timelines
* Estudar timing, easing e fluidez das animações
* Explorar impactos de performance ao combinar animações com scroll customizado

## Stack Tecnológica

* **Next.js** – Estrutura da aplicação e roteamento
* **GSAP** – Controle de animações e motion
* **Locomotive Scroll** – Smooth scroll e eventos de rolagem

## Como Executar

Clone o repositório e instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

## Observações

* Este é um **projeto de estudo e experimentação**, não voltado para produção.
* O conteúdo é propositalmente simples para manter o foco no comportamento das animações.
* Ideal como sandbox para testar ideias de motion design e interações baseadas em scroll.
